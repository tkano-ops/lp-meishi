## 1. 事業コンセプト

### ワンフレーズ
**「かざすだけで、あなたのストーリーが伝わる名刺」**

### 課題（ペインポイント）
- 交流会で名刺交換しても、後日見返したとき **「あの人誰だったっけ」** と思い出せない
- 紙の名刺では「何をしている人か」「どんな実績があるか」が伝わりきらない
- 短い会話では **自分のことをうまく伝えきれない**
- 名刺入れに埋もれ、商談・案件に繋がらない
- SNS・Web・ポートフォリオなど情報が分散している

### 解決策
NFC カード／QRコード付きカードをスマホにかざすだけで、
**完全オーダーメイドのパーソナルLP（ランディングページ）** が即座に表示される。

短時間で「この人に仕事を頼みたい」「もう一度会いたい」と思わせる
**印象に残るデジタルプロフィール** を提供する。

### 4つの提供価値
1. **出会いを忘れないものに** — 一期一会の出会いを繋がりに変える
2. **その人らしさを伝える** — 一人ひとりのストーリー・個性を大切に
3. **第一印象を変える** — ビジネスの成果は印象で決まる
4. **自分を売るを簡単に** — セルフブランディングを民主化

## 2. 事業体制

### 運営形態
- **別会社を設立**（グッドキャストとは別法人）
- **パートナーと共同経営**（対等な関係で全業務を分担）
- 制作はClaude Codeを活用して効率化

### 役割分担
| 担当 | 範囲 |
|---|---|
| 加納 + パートナー（対等） | 営業・制作・企画・運営すべて |
| Claude Code | LP制作の技術的な実装を支援 |
| 外注（必要に応じて） | プロ撮影、動画制作など |

## 4. プロダクト詳細

### LP（ランディングページ）に含まれるセクション

```
┌─────────────────────────────────────┐
│  ヘッダー（写真 + キャッチコピー）     │
├─────────────────────────────────────┤
│  自己紹介・ストーリー                 │
│  （経歴、なぜこの仕事をしているか）     │
├─────────────────────────────────────┤
│  事業内容・サービス一覧               │
│  （何を提供できるか）                 │
├─────────────────────────────────────┤
│  実績・メディア出演                   │
│  （TV、雑誌、登壇、受賞歴など）        │
├─────────────────────────────────────┤
│  お客様の声・推薦                     │
│  （権威性・信頼性の強化）              │
├─────────────────────────────────────┤
│  ビジョン・今後の展望                 │
│  （共感・応援を生む）                 │
├─────────────────────────────────────┤
│  SNSリンク集                         │
│  （X, Instagram, YouTube, note等）    │
├─────────────────────────────────────┤
│  お問い合わせ・予約ボタン             │
│  （LINE, メール, Calendly等）         │
└─────────────────────────────────────┘
```

### デザインの方針
- **お客様ごとにブランディング・好みに合わせて変える**（テンプレート固定ではない）
- シンプル・洗練 / ダーク・高級感 / カラフル・ポップ など柔軟に対応
- 完全オーダーメイドがこのサービスの核

### 物理カード（NFCカード付きで提供）
- **NFC内蔵カード** — スマホをかざすだけでLP表示
- **QRコード** — NFC非対応端末でも対応
- デザインはオーダーメイド（ブランドカラー、ロゴ反映）
- 素材：マット／光沢／金属調など選択可能

### 技術構成（想定）
| 要素 | 技術 |
|---|---|
| フロントエンド | Next.js (App Router) |
| ホスティング | Vercel |
| CMS / データ管理 | Notion API or Supabase |
| ドメイン | サブドメイン形式（例: taro.lp-meishi.jp）|
| NFC | NTAG215 / NTAG216チップ |
| アクセス解析 | カード別の閲覧数トラッキング |
| LP制作 | Claude Code で効率化 |

## 9. サービス名（決定）

### ヒトイロ（HITOIRO / 人色）
**キャッチコピー: 「あなたの色を、一枚に。」**

#### 由来・コンセプト
- **人色** = 一人ひとり違う色（個性）を、そのままカードに移す
- 「十人十色」「色とりどり」という日本人に馴染んだ言葉から派生
- **完全オーダーメイド = その人の色で作る**、という事業の核を一語で表現
- カケハシ（goodcastの日本語学習アプリ）と同じ「カタカナ和語×キャッチコピー」構造で、ブランドファミリーの統一感を出す

#### ロゴ表示
```
ヒトイロ  by goodcast
HITOIRO
あなたの色を、一枚に。
```

#### ドメイン候補（法人設立後に取得検討）
- `hitoiro.co.jp` — 2026/05/31 削除予定、再登録可能になったタイミングで申請
- `hitoiro.io` — 空き、即取得可能（テック系・モダン）
- ※ `hitoiro.com` `hitoiro.jp` `hitoiro.net` は既に他者が保有
